日常笔记-分类整理res目录下Layouts中的布局文件

发表于 2019-09-08

在moudle下的build.gradle文件中的android节点下配置以下代码可以分类整理res/layout中的资源目录：

sourceSets {
    main {
        jniLibs.srcDirs = ['libs']
        res.srcDirs =
                [
                        'src/main/res/layouts/activity',
                        'src/main/res/layouts/fragment',
                        'src/main/res/layouts/dialog',
                        'src/main/res/layouts/item',
                        'src/main/res/layouts/view',
                        'src/main/res/layouts/include',
                        'src/main/res/layouts',
                        'src/main/res'
                ]
    }
}

切记’src/main/res/layouts’,和’src/main/res’要保持在最后一行。

这里一定要记录一个不经常遇到，但是会忘记如何解决的问题，如下图：

就是layouts展示目录不完整，有些子目录没展示出来，通常更新完AS的版本后会出现这个问题。

解决办法是，取消勾选空文件夹过滤即可。
